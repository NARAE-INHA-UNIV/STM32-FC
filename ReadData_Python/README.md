# FC 데이터 읽기

## 실행 방법

1. 패키지 설치 (최초 1회만)

```bash
pip isntall pyserial questionary
```

2. 실행

```bash
python main.py
```

3. FC가 연결된 포트 선택 (화살표 키로 이동, Enter로 선택)

```bash
Choose Serial Port: (Use arrow keys)
   /dev/cu.debug-console - n/a
   /dev/cu.wlan-debug - n/a
 » /dev/cu.Bluetooth-Incoming-Port - n/a
```

4. 결과 확인
