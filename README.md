# STM32 FC


![pcb img](./STM32-FC.jpg)


인하대학교 모형항공기 동아리 나래에서 2024년 동계 UAV 활동으로 진행한 프로젝트임. <br>
회로와 펌웨어를 개발하되, 기성 pixhawk 모듈과 호환성을 가지기 위해 pixhawk 6C mini의 인터페이스와 흡사하게 제작하는 것을 목표로함. <br>

- 활동명 : [UAV] FC 제작 활동
- 일시 : 2024 동계 방학 - 진행중
- 팀명 :  항공전자 (구 FC)
- 팀장 : 정영우(2025), 이현제(2024)
